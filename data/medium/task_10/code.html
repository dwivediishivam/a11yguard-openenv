Installation Guide

Last updated: March 15, 2025

This guide walks you through installing DocHub SDK in your project. The SDK supports Node.js 18+ and Python 3.10+.

Prerequisites

Make sure you have node version 18 or higher installed. You can verify with:

$ node --version
v18.17.0

Install via npm

Run the following command in your project directory:

$ npm install @dochub/sdk

FAQ

Can I use DocHub with TypeScript? ▼
Yes! DocHub SDK ships with full TypeScript type definitions. No additional @types package needed.
What Node.js versions are supported? ▼
DocHub SDK requires Node.js 18 or higher. LTS versions are recommended for production use.
Is there a Python SDK? ▼
Yes, install it with: pip install dochub-sdk. Python 3.10+ is required.
